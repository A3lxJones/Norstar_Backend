# Backend (Norstar)

Minimal TypeScript + Express scaffold for the Norstar backend.

Files created:

- `src/index.ts` - entry point (starts server)
- `src/app.ts` - express app setup
- `src/routes/` - routes
- `src/controllers/` - controllers
- `src/types/` - shared types
- `src/data/` - temporary mock data

Quick start (from repo root):

1. cd backend
2. npm install
3. npm run dev
